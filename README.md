# 💄 Clube Revele — Sistema de Assinaturas

Sistema de clube de assinaturas desenvolvido para a **Revele Cosméticos**. A proposta é permitir que clientes assinem o **Clube Revele** e tenham descontos exclusivos nos produtos — tanto nas compras online quanto na loja física — além de poderem comprar produtos avulsos pelo site.

> Projeto pessoal, em desenvolvimento. O back-end já está em estágio avançado; o front-end está em construção.

---

## ✨ Funcionalidades

**✅ Implementado (back-end)**
- Autenticação de usuários
- Criação e gerenciamento de contas
- Gerenciamento de assinaturas do Clube Revele
- Cadastro e gerenciamento de produtos
- Criação de pedidos

**🚧 Em desenvolvimento**
- Métodos de pagamento
- Interface do usuário (front-end)
- Refinamentos gerais

---

## 🛠️ Tecnologias

**Back-end:** Java · Spring Boot
**Front-end:** TypeScript *(em desenvolvimento)*
**Banco de dados:** PostgreSQL

---

## 📋 Sobre o projeto

O Clube Revele funciona de forma parecida com clubes de assinatura conhecidos: o usuário paga por uma assinatura e passa a ter acesso a preços reduzidos nos produtos. O diferencial é a integração entre o ambiente online e a loja física — o desconto de assinante vale nos dois.

O usuário pode:
- Comprar uma assinatura do clube e obter descontos
- Comprar produtos avulsos diretamente pelo site
